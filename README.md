# Football Team Bot ⚽

MVP Telegram-бота для футбольной группы:

- FIFA-профиль игрока
- позиция и характеристики
- отдельное поле "умение играть в воротах"
- готовность стоять на воротах
- запись на игру через кнопки
- автоматическое деление на 2 команды
- учет основных и запасных вратарей

## 1. Установка

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## 2. Настройка

Скопируйте пример переменных:

```bash
cp .env.example .env
```

Заполните:

```env
API_ID=...
API_HASH=...
BOT_TOKEN=...
ADMIN_IDS=...
```

- `BOT_TOKEN` берется у `@BotFather`
- `API_ID` и `API_HASH` берутся на `my.telegram.org`
- `ADMIN_IDS` — Telegram ID админов через запятую. Можно оставить пустым, если бот добавлен в группу и может проверять админов группы.

## 3. Запуск

```bash
python bot.py
```

## 4. Команды

### Игрок в личке бота

```text
/start
/new_profile
/profile
/cancel
```

### Админ в группе

```text
/create_match
/players_today
/balance
/shuffle
/reset_match
```

## 5. Позиции

```text
GK   — вратарь
DEF  — защитник
MID  — полузащитник
FWD  — нападающий
WING — вингер
```

## 6. Готовность играть в воротах

```text
0 — никогда
1 — только если нет вратаря
2 — иногда
3 — могу играть
4 — я вратарь
```

## 7. Как работает баланс

1. Бот сначала ищет 2 лучших варианта для ворот:
   - основной вратарь
   - дополнительная позиция GK
   - высокий `goalkeeper_skill`
   - высокая готовность стоять в воротах

2. Потом разводит их по разным командам.

3. Остальных игроков распределяет по:
   - общему рейтингу
   - атаке
   - защите
   - скорости
   - позициям

4. `/shuffle` дает другой вариант, но всё равно пытается сохранить баланс.
